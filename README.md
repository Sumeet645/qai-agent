# QAI Agent — AI-Powered Test Case Generator

An AI-powered web application for QA Engineers and SDETs 
that automatically generates structured test cases from 
plain English requirements.

## Features
- Generate test cases from any requirement in seconds
- Structured output with TC_ID, Type, Steps, Expected Result, Priority
- Color coded — Green (Positive), Red (Negative), Orange (Edge Cases)
- Built with Claude AI + Flask

## Tech Stack
- Python + Flask
- Anthropic Claude API
- HTML/CSS

## How to Run
```bash
pip install anthropic flask
export ANTHROPIC_API_KEY=your-key-here
python3 app.py
```
Open browser: http://127.0.0.1:5000
